Detailed Description

TCP server that accepts browser connections and manages the intercept queue.

ProxyServer listens on a configurable address/port (default 127.0.0.1:8080) and creates a ProxyConnection for every accepted socket. When interception is enabled it serialises incoming requests through an internal queue so that the UI can inspect and modify them one at a time.
